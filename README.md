

## ETAPA 1 — Entendimiento del dataset

Objetivo: saber qué hay en el dataset y como está organizado

Acciones:

- ver primeras filas
- listar columnas
- identificar tipos de datos
- detectar nulos
- entender qué representa cada variable

Preguntas guía:

- ¿qué es cada fila?
- ¿qué columnas son clave?
- ¿qué problemas veo?


## ETAPA 2 — Limpieza de datos

Objetivo: dejar el dataset listo para análisis

Acciones:

- eliminar duplicados
- tratar nulos (eliminar o imputar)
- convertir fechas (date_added)
- separar columnas útiles (ej: géneros)
- estandarizar nombres


## ETAPA 3 — Definir preguntas (ANTES del EDA)

Objetivo: Definir 3–5 preguntas:

Ejemplo Netflix:

- ¿Cómo ha crecido el contenido con el tiempo?
- ¿Qué tipo domina: Movies o TV Shows?
- ¿Qué países producen más contenido?
- ¿Cuáles son los géneros más comunes?


## ETAPA 4 — EDA (Exploratory Data Analysis)

Objetivo: responder preguntas

Acciones:

- conteos
- agrupaciones
- estadísticas básicas

Ejemplos:

contenido por año
contenido por tipo
top países
top géneros


## ETAPA 5 — Visualización

Objetivo: comunicar mejor

Ejemplos:

gráficos de barras → categorías
líneas → evolución temporal


## ETAPA 6 — Insights (LO MÁS IMPORTANTE)

Objetivo: interpretar

Hacer 3–5 insights claros.


## ETAPA 7 — Outputs

Guardar:

gráficos → outputs/images/
insights → outputs/insights.md


## ETAPA 8 — README (final)

Resúmen:

- objetivo
- preguntas
- insights clave
- herramientas
